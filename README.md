**Project Title**

Spider Robot with Obstacle Avoidance

**📌 Project Overview**

This project involves designing and building a 4-legged spider robot that can walk using coordinated servo motor movements. The robot is controlled using an ESP32 microcontroller and a PCA9685 servo driver to manage multiple servos efficiently. It uses an ultrasonic sensor to detect obstacles in front and responds by stopping or changing direction. The project combines mechanical design, electronics, and programming to create an interactive robotic system.
